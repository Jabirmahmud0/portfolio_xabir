# ElegantPortfolio

A modern, elegant, and interactive portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

## Features
- **Light/Dark Theme Switcher**
- **Spotlight Mouse Effect**
- **Section Navigation with Scroll Tracking**
- **Animated UI with Framer Motion**
- **Project, Experience, and Blog Sections**
- **Random Quote Generator**
- **Responsive Design**

## Getting Started

### Prerequisites
- Node.js (v16+ recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd <your-repo-directory>
   ```
2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```
3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```
4. Open [http://localhost:5173](http://localhost:5173) to view the app.

## Customization
- Update your name, bio, projects, experience, and blog posts in `src/App.jsx`.
- Tailwind CSS classes are used for styling. Adjust as needed for your brand.
